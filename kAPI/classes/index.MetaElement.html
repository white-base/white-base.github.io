    • OptionalvOpt: number

      직렬화 옵션입니다. 기본값은 0입니다.

      • 0: 참조 구조 (_guid: 예, $ref: 예)
      • 1: 중복 구조 (_guid: 예, $ref: 예)
      • 2: 비참조 구조 (_guid: 아니오, $ref: 아니오)
    • Optionalowned: object | object[]

      소유하는 상위 객체들입니다. 기본값은 빈 객체입니다.

    Returns object
