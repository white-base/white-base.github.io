    key: string;
    type: string;
}

셀렉터를 정의합니다. type

  • val 또는 value: 요소의 value 속성값
  • text: 요소의 텍스트값
  • html: 요소의 HTML 값
  • css.속성명: CSS의 속성값 (객체)

  • attr.속성명: 요소의 속성명값 (현재 상태)
  • none: 아무 작업도 수행하지 않음, 표현의 목적 예시: 'value', 'text', 'css.color', 'prop.disabled'
